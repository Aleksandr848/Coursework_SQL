    Требования к курсовому проекту:
        1. Составить общее текстовое описание БД и решаемых ею задач;

        2. минимальное количество таблиц - 10;

        3. скрипты создания структуры БД (с первичными ключами, индексами, внешними ключами);

        4. создать ERDiagram для БД;

        5. скрипты наполнения БД данными;

        6. скрипты характерных выборок (включающие группировки, JOIN'ы, вложенные таблицы);

        7. представления (минимум 2);

        8. хранимые процедуры / триггеры;

    1. База данных allbooks для хранения и сортировки данных для сайта по продаже бумажных, электронных и аудио книг.
Задачи базы данных - сортировка по названию, автору, типу книги, рейтингу, стоймости, тэгам, жанру и выборка по
нескольким критериям.

    2. Основные сущности базы данных allbooks - это пользователи (users) и книги (books)

USERS - для регистрации достаточно email и пароль. Вся дополнительная информация в таблице profiles.
Таблица PROFILES содержит следующие данные:
    first_name, last_name, phone_number, gender, birthday, about_me, payment_method.

Таблица BOOKS содержит следующие данные:
    author, title, type, language, description, age_limit, date_of_writing, translator

Вспомогательные таблицы:

    audiobook - длительность, чтец, доступность
    electronic_book формат, доступность
    paper_book - обложка, издательство, количество страниц, количество книг на складе;
    readers - имя, фамилия
    authors - имя, фамилия, год рождения, год смерти, страна, пол, жанр;
    book_type - бумажная, электронная, аудио;
    rating - пользователь --> книга от 1 до 5;
    reviews - пользователь --> книга;
    quotes - цытаты
    likes - пользователь --> reviews или quotes;
    my_books - пользователь --> купленные книги, отложенные книги, просмотренные книги, корзина.
    countries - список стран
    languages - список языков
    tags - список тэгов
    genre - список жанров
    books_tag - таблица книг с тэгами
    books_genre - таблица книг с жанрами
    users_purchases - таблица покупок пользователя
    publishing_houses - таблица издательств
    books_publishing_house - таблица принадлежности книг к издательствам
